This version provides several enhancements and addresses items discovered in client-side testing. 

Poison Center feedback is critically important to ensure we can enhance Toxicall to meet the needs of all poison centers.

### Enhancements

\# 1956 Drag and Drop Reordering added for most lookup lists.
\# 1970 User Selected Grid Colors now has an Apply button to view the changes before closing the configuration window.
\# 1976 Added application default of "New Case Status" which can be changed for each Toxicall installation. The default setting is "Quick Saved"



### Bug Fixes

\# 1961 Addressed Sync issue with Backward Migration. This was causing a problem with empty system table items.
\# 1964 Updated the Migration Alerts to properly reflect when Migration is offline or not configured.
\# 1978 Resolved an issue of the Sync Status color not changing when Sync comes online/offline.
\# Employee removed the requirement for the email address.
