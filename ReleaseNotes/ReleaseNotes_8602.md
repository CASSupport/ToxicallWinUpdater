Additional significant updates in this release. Updates to the Search Filters (Search Criteria) Added new Filter Ribbons (Quick Search Toolbar) allowing easy access to saved search criteria.


### Search Filters
Formerly known as Search Criteria, you can now save and load search criteria for easy access to case data.
* Early Preview

We are still working on the Filter Criteria, allowing filters to prompt for entry of data elements like dates.


### Filter Ribbons
Formerly known as Quick Search Toolbar, there is now a new Filters ribbon bar allowing single-click access to search filters. Create your own Ribbon Bar to make the most important search criteria available for your needs.
* Early Preview


### View Variants (working name)

Formerly known as "Displays" this will allow you to save your data list (grid) layout designs for future use. This will be helpful for both case management as well as data export needs. Choose both the data columns and sort order of information display. 
* Early Preview


We are building [search filter documentation](https://toxicalldemo.com/docs/tox5/search-filters-overview/) which will be updated as we fine-tune the functionality.
