### Known Issues

When attempting to use the grid style direct entry for Routes, Clinical Effects, Therapies, or Scenarios, Error 1044 is presented. Users can press OK and continue without issue. Until the problem is resolved, users should use the popup window to enter these data. The popup window is accessed right-clicking and selecting "Open Popup" or by clicking the orange notepad button with the green + symbol in the grid's toolbar.
