This version brings the first implementation of the new Search Filters capabilities. This will allow users to save search filters (criteria) using an all new builder. These filters are applied to the case list initially. Future updates will allow applying the search filters to reports and dashboards. Extensive filter criteria can be designed, saved and modifies. Our team will be building base search filters which will be distributed to all toxiCALL® users using FORM. The same drop-down menu functionality is also used to build the case validation rules.

We are building [search filter documentation](https://toxicalldemo.com/docs/tox5/search-filters-overview/) which will be updated as we fine-tune the functionality.

Additional enhancements have been added to the Lab Module.
