Great new features include Toxinz integration, Micromedex Integration enhancement, and Favorites list for easy access to frequently used items.

Integration with Toxinz for substance lookup. This is the first draft, with additional features to come later. Integration of the substance description and product code will be in a future release as we are working with Toxinz for integration features to be developed.

Update to the Micromedex substance integration. Product details are now stored in a formatted view for case review.

This version provides several enhancements and addresses items discovered in client-side testing. 

Poison Center feedback is critically important to ensure we can enhance Toxicall to meet the needs of all poison centers.

### Enhancements

\# 1998 Favorites List is now available in the left Sidebar. This allows easy access to items that are frequently used. Use the new 'Add View to Favorites' Action item in the toolbar to add or remove from favorites. You can add List views such as Case List, Free Areas, Etc. Even add specific cases that you are following. You can also add Reports and Dashboards using the 'Favorite Dashboard/Report Management' action button in the ribbon bar to select which reports or dashboards to include. Documentation is coming soon.
\#2011 Toxinz integration is now ready for client usage and feedback. We have limited integration capabilities from the Toxinz API and we are working with their team for future enhancements. A PDF will now be stored for each substance of the product details added to the case.
\#2011 Micromedex integration is now enhanced to store product details in a PDF, which can be viewed for each substance. In the future, poison centers can disable this feature if additional data storage is not desired.
