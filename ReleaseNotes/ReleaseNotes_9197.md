Enhancements include audit tracking, security role management, additional fields for Patient ID and CAS Number, Healthcare Facility Import, and more.


Poison Center feedback is critically important to ensure we can enhance Toxicall to meet the needs of all poison centers.

### Enhancements

\# toxiCALL created Security Roles are now released through System Tables. Internal roles can also be created
\# Case and Employee Audit Trails have been enhanced for additional tracking details
\# Custom Validation Rules be refined to better accommodate non-toxiCALL 4 data migration
\# New Import module supporting import of Healthcare Facilities and another data lists
\# New Patient ID field can be used in addition to the EMR Number
\# New CAS Number can be added for each substance


### Bug Fixes

\# Corrected issue with DOB calculation when the date is not changed



### Known Issues

\# Password reset not yet working properly through CAS Server
\# Import module may require additional testing and refinement
